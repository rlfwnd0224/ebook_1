# 🎮 동화책 학습 게임 기획서

## 📚 개요
각 동화책마다 **8개의 학습 단어**를 선정하고, 이를 활용한 **3가지 교육 게임**을 제공합니다.

---

## 🎯 학습 단어 시스템

### 데이터 구조
```json
{
  "id": "story-001",
  "title": "백설공주",
  "learningWords": [
    {
      "word": "왕비",
      "meaning": "왕의 아내",
      "example": "새로운 왕비가 궁전에 도착했어요.",
      "imageUrl": "https://example.com/queen.jpg"
    },
    {
      "word": "거울",
      "meaning": "자신의 모습을 비춰볼 수 있는 물건",
      "example": "왕비는 마법 거울에게 물어봤어요.",
      "imageUrl": "https://example.com/mirror.jpg"
    }
    // ... 총 8개
  ]
}
```

### 백설공주 학습 단어 예시
1. **왕비** - 왕의 아내
2. **거울** - 모습을 비추는 물건
3. **사과** - 빨갛고 둥근 과일
4. **난쟁이** - 키가 작은 사람
5. **독약** - 몸에 해로운 약
6. **숲** - 나무가 많은 곳
7. **성** - 왕과 공주가 사는 집
8. **마법** - 신비한 힘

## 🎮 게임 1: 단어 카드 매칭 (Memory Match)

### 게임 설명
- 8개 단어 + 8개 그림 = 총 16장의 카드
- 카드를 뒤집어서 단어와 그림을 매칭하는 게임

### UI/UX 디자인
```
┌─────────────────────────────────────┐
│  🎮 단어 카드 매칭 게임              │
│  ⏱️  시간: 0:45  │  ⭐ 점수: 3/8    │
├─────────────────────────────────────┤
│                                     │
│   [?]  [?]  [?]  [?]               │
│                                     │
│   [?]  [?]  [?]  [?]               │
│                                     │
│   [?]  [?]  [?]  [?]               │
│                                     │
│   [?]  [?]  [?]  [?]               │
│                                     │
└─────────────────────────────────────┘
```

### 기능
- 카드 뒤집기 애니메이션 (flip animation)
- 2장 선택 시 자동 비교
- 맞으면: ✅ 고정 + 효과음
- 틀리면: ❌ 다시 뒤집기
- 완료 시: 🎉 축하 메시지 + 시간 기록

### 난이도 조절
- **쉬움**: 4쌍 (8장) - 3-4세
- **보통**: 6쌍 (12장) - 5-6세
- **어려움**: 8쌍 (16장) - 7세+

## 🎮 게임 2: 단어 퀴즈 (Word Quiz)

### 게임 설명
- 단어의 뜻을 보고 정답 선택하기
- 4지선다형 문제

### UI/UX 디자인
```
┌─────────────────────────────────────┐
│  📝 단어 퀴즈                        │
│  문제 3/8  │  ⭐ 정답: 2개           │
├─────────────────────────────────────┤
│                                     │
│  💡 질문: 왕의 아내를 뭐라고 하나요? │
│                                     │
│  ┌─────────────┐  ┌─────────────┐ │
│  │ A. 공주    │  │ B. 왕비    │ │
│  └─────────────┘  └─────────────┘ │
│                                     │
│  ┌─────────────┐  ┌─────────────┐ │
│  │ C. 난쟁이  │  │ D. 거울    │ │
│  └─────────────┘  └─────────────┘ │
│                                     │
└─────────────────────────────────────┘
```

### 기능
- 랜덤 단어 순서
- 4개 선택지 (정답 1개 + 오답 3개)
- 정답: ✅ 녹색 표시 + 칭찬 메시지
- 오답: ❌ 빨간색 표시 + 정답 알려주기
- 진행 바 표시
- 최종 점수: 8개 중 맞춘 개수

### 문제 유형
1. **뜻 맞히기**: "왕의 아내를 뭐라고 하나요?" → 왕비
2. **설명 맞히기**: "빨갛고 둥근 과일은?" → 사과
3. **그림 보고 맞히기**: [사과 그림] → 사과

## 🎮 게임 3: 단어 만들기 (Word Builder)

### 게임 설명
- 흩어진 글자를 순서대로 조합하여 단어 완성하기
- 드래그 앤 드롭 방식

### UI/UX 디자인
```
┌─────────────────────────────────────┐
│  ✏️ 단어 만들기                      │
│  문제 2/8  │  힌트: 왕의 아내        │
├─────────────────────────────────────┤
│                                     │
│  💭 힌트 이미지                      │
│  [👸 왕비 그림]                      │
│                                     │
│  📝 만들 단어:                       │
│  ┌───┐ ┌───┐                       │
│  │   │ │   │                       │
│  └───┘ └───┘                       │
│                                     │
│  🔤 글자 선택:                       │
│  ┌───┐ ┌───┐ ┌───┐ ┌───┐         │
│  │ 왕 │ │ 비 │ │ 거 │ │ 울 │     │
│  └───┘ └───┘ └───┘ └───┘         │
│                                     │
│  [확인하기] 버튼                     │
│                                     │
└─────────────────────────────────────┘
```

### 기능
- 글자 드래그 앤 드롭
- 모바일: 터치 방식으로 글자 선택
- 정답 확인 버튼
- 정답: ✅ 축하 + 다음 문제
- 오답: ❌ 다시 시도 (힌트 추가)
- 힌트: 그림 + 뜻 설명

### 난이도 조절
- **쉬움**: 2글자 단어 (4개 선택지)
- **보통**: 3글자 단어 (6개 선택지)
- **어려움**: 4글자 단어 (8개 선택지)

## 🎨 전체 UI/UX 가이드라인

### 색상 테마
- **메인 색상**: `#FF6B9D` (핑크) - 친근하고 따뜻한 느낌
- **서브 색상**: `#FFA500` (오렌지) - 활발하고 재미있는 느낌
- **배경**: `#FFF9F0` (크림) - 눈에 편안한 색
- **정답**: `#4CAF50` (녹색)
- **오답**: `#FF5252` (빨간색)
- **텍스트**: `#333333` (진한 회색)

### 폰트
- **제목**: Noto Sans KR Bold, 24px-32px
- **본문**: Noto Sans KR Regular, 18px-20px
- **버튼**: Noto Sans KR Medium, 16px-18px

### 아이콘
- **게임 아이콘**: 🎮 🎯 ✏️ 🎨
- **피드백 아이콘**: ✅ ❌ ⭐ 🎉 💯
- **힌트 아이콘**: 💡 💭 🔍

### 애니메이션
- **버튼 호버**: Scale 1.05, 0.2s ease
- **카드 뒤집기**: RotateY 180deg, 0.6s
- **정답**: Bounce + Confetti 효과
- **오답**: Shake 효과
- **페이지 전환**: Fade in/out, 0.3s

### 반응형 디자인
- **모바일** (≤768px): 
  - 1열 레이아웃
  - 버튼 크기 증가 (최소 44px)
  - 터치 친화적 간격
- **태블릿** (769px-1024px):
  - 2열 레이아웃
  - 적절한 여백
- **데스크톱** (>1024px):
  - 3-4열 레이아웃
  - 최대 너비 1200px

## 🎯 학습 효과

### 교육적 목표
1. **어휘력 향상**: 새로운 단어 학습
2. **기억력 강화**: 반복 학습을 통한 기억력 증진
3. **집중력 향상**: 게임을 통한 재미있는 학습
4. **성취감**: 게임 완료 시 보상과 칭찬

### 학습 단계
1. **동화 읽기** → 자연스럽게 단어 노출
2. **게임 1 (매칭)** → 단어와 이미지 연결
3. **게임 2 (퀴즈)** → 단어의 뜻 이해
4. **게임 3 (만들기)** → 단어 구조 이해

## 📱 기술 구현 스택

### 프론트엔드
- **HTML5 Canvas** (게임 2, 3) 또는 **DOM 기반**
