1. 프로젝트 개요

1.1 프로젝트 목표

유아 교육을 위한 인터랙티브 동화책 + 학습 게임 통합 플랫폼 구축

  • 직관적이고 아이들이 쉽게 사용할 수 있는 UI/UX
  • 이미지와 텍스트가 결합된 동화책 뷰어
  • TTS(Text-to-Speech) 음성 읽기 기능
  • 🆕 학습 단어 기반 교육 게임 시스템
  • 관리자 페이지를 통한 쉬운 콘텐츠 관리
  • 모바일 반응형 디자인

1.2 핵심 가치

  • 접근성: 웹 기반으로 어디서나 접근 가능
  • 교육성: TTS + 학습 게임을 통한 종합 학습 지원
  • 재미: 게임화된 학습으로 몰입도 향상
  • 확장성: 관리자 페이지로 쉽게 콘텐츠 추가
  • 안정성: JSON 기반 데이터 관리로 간단하고 안정적

1.3 v5.0 주요 업데이트

🆕 새로운 기능
  • ✓ 학습 단어 시스템: 각 동화책마다 8개 학습 단어 지정
  • ✓ 단어 카드 매칭 게임: 3단계 난이도, 타이머, 점수 시스템
  • ✓ 게임 선택 화면: 학습 단어 미리보기 + 게임 메뉴
  • ✓ 학습 단어 관리: 관리자 페이지에서 학습 단어 입력/수정
  • ✓ 타임키즈_세종대왕: 역사 교육 콘텐츠 추가 (30페이지)

No. 단어 뜻 예문

7 성 왕과 공주가 사는 큰 집 왕자의 성에서 결혼식이 열렸어요.
8 마법 신비한 힘으로 이상한 일을 하는 것 마법 거울은 모든 것을 알고 있었어요.

구분 기술 비고

배포 PM2 (개발) / Vercel (프로덕션) 간편한 배포
버전 관리 Git + GitHub 코드 관리 및 협업

4. 핵심 기능

4.1 사용자 페이지 (기존 기능)

✅ 메인 페이지 (index.html)
  • ✓ 동화책 목록 카드 형태로 표시
  • ✓ 연령대별 필터링
  • ✓ 태그별 필터링 및 검색
  • ✓ 반응형 그리드 레이아웃
✅ 동화책 뷰어 (reader.html)
  • ✓ 이미지 + 텍스트 동시 표시
  • ✓ 이전/다음 페이지 네비게이션
  • ✓ 키보드 단축키 (← → 스페이스바)
  • ✓ TTS 음성 읽기 (재생/일시정지/속도 조절)
  • ✓ TTS 컨트롤 토글 (모바일 최적화)
  • ✓ 읽기 진행률 저장 (LocalStorage)
  • ✓ 🆕 학습 게임 버튼 - 게임으로 바로 이동

4.2 학습 게임 시스템 (🆕 신규 기능)

🆕 게임 선택 화면 (games.html)
  • ✓ 학습 단어 미리보기: 8개 단어 뱃지 표시
  • ✓ 게임 메뉴:
    • 🃏 단어 카드 매칭 (활성)
    • 📝 단어 퀴즈 (준비 중)
    • ✏️ 단어 만들기 (준비 중)
  • ✓ 동화책으로 돌아가기 버튼
🆕 단어 카드 매칭 게임 (memory-match.html)
  • ✓ 3단계 난이도:
    • 쉬움: 4쌍 (8장) - 3-4세
    • 보통: 6쌍 (12장) - 5-6세
    • 어려움: 8쌍 (16장) - 7세+
  • ✓ 게임 기능:
    • 카드 뒤집기 (Opacity 애니메이션)
    • 타이머 (실시간 시간 측정)
    • 점수 시스템 (매칭 성공 카운트)
    • 시도 횟수 기록
  • ✓ 완료 화면:
    • 🎉 축하 메시지
    • ⭐ 별점 시스템 (1-3개)
    • 🏅 메달 (금/은/동)
    • 📊 통계 (시간/시도/점수)
    • 다시하기 / 게임 선택 버튼
  • ✓ 모바일 반응형
  • ✓ 이미지 로딩: 유효성 검사 + 로드 완료 확인

4.3 관리자 페이지

✅ 동화책 관리 (admin.html)
  • ✓ 동화책 생성/수정/삭제
  • ✓ 통합 편집 화면 (이미지 + 텍스트 동시 조회/수정)
  • ✓ 기존 데이터 자동 로드
  • ✓ 페이지별 미리보기
  • ✓ 🆕 학습 단어 설정 버튼
🆕 학습 단어 관리 (learning-words.html)
  • ✓ 8개 단어 입력 폼:
    • 단어 (한글)
    • 뜻 (간단한 설명)
    • 예문 (문장 형태)
    • 이미지 URL (동화책 이미지 재사용 가능)
  • ✓ 자동 로드: 기존 학습 단어 자동 불러오기
  • ✓ 이미지 미리보기: URL 입력 시 미리보기
  • ✓ 저장 API: PUT /api/admin/storybooks/:id/learning-words

5. API 엔드포인트

5.1 공개 API

Method Endpoint 설명
GET /api/storybooks 전체 동화책 목록 조회
GET /api/storybooks/:id 특정 동화책 상세 조회

5.2 관리자 API

Method Endpoint 설명
POST /api/admin/storybooks 새 동화책 생성
PUT /api/admin/storybooks/:id 동화책 정보 수정
DELETE /api/admin/storybooks/:id 동화책 삭제
POST /api/admin/storybooks/:id/pages/bulk-images 다중 이미지 URL 추가/교체
POST /api/admin/storybooks/:id/pages/bulk-texts 텍스트 일괄 입력
PUT /api/admin/storybooks/:id/learning-words 🆕 학습 단어 저장

5.3 페이지 라우트

URL 페이지 설명
/ index.html 메인 페이지
/reader/:id reader.html 동화책 뷰어
/admin admin.html 관리자 페이지
/admin/learning-words learning-words.html 🆕 학습 단어 설정
/games games.html 🆕 게임 선택
/games/memory-match memory-match.html 🆕 단어 카드 매칭 게임

6. 학습 게임 UI/UX 가이드

6.1 색상 테마

  • 메인 색상: #FF6B9D (핑크) - 친근하고 따뜻한 느낌
  • 서브 색상: #FFA500 (오렌지) - 활발하고 재미있는 느낌
  • 배경: #FFF9F0 (크림) - 눈에 편안한 색
  • 정답: #4CAF50 (녹색)
  • 오답: #FF5252 (빨간색)

6.2 애니메이션

  • 카드 뒤집기: Opacity 전환, 0.3s
  • 정답: Scale + 녹색 테두리
  • 오답: Shake 효과
  • 페이지 전환: Fade in/out

6.3 반응형 디자인

  • 모바일 (≤768px): 4×2 또는 3×2 그리드
  • 태블릿/데스크톱: 4×4 그리드
  • 카드 크기: 자동 조절 (최소 100px)
